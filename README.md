# react-micro-frontend-setup

## Installation
Install the dependencies and devDependencies and start the server.

### Step 1
```sh
cd react-micro-frontend-setup
cd footer-mf
npm install
npm run start:live
```
### Step 2
```sh
cd react-micro-frontend-setup
cd header-mf
npm install
npm run start:live
```
### Step 3
```sh
cd react-micro-frontend-setup
cd mainApp
npm install
npm run start:live
```
